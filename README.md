## About

Jumphigh is a simple platformer game inspired by Icy Tower. 
It's made with use of c++17 and SFML.

#### Screenshot:

![Screenshot](/resources/Screenshots/Screenshot.png)

## Requirements
```sh
- sfml
- g++
- make
```

## Compiling (Linux)

1. Clone the repo
```sh
git clone https://github.com/4rlenrey/JumpHigh.git
```
2. Compile 
```sh
make
```
2. Run 
```sh
./Jumphigh
```
